---
name: Naming always in English
description: All code naming (variables, functions, files, hooks, types) must be in English
type: feedback
---

All naming in code must be in English — variables, functions, hooks, types, file names, constants.

**Why:** Consistency and professionalism. Spanish is reserved for user-facing strings (UI text, error messages).

**How to apply:** Never use Spanish words in identifiers. E.g., `useSupermarkets` not `useSupermercados`, `ShoppingItem` not `ArticuloCompra`.
